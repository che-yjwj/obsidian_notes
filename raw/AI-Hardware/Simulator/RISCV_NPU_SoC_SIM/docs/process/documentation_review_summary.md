본 문서는 더 이상 상세 리뷰 본문을 담는 문서가 아니다.  
현재 역할은 과거 리뷰 사이클이 존재했다는 사실과, 그 결과가 어디로 흡수되었는지를 안내하는 **운영용 포인터**다.

## Usage Rule

- 이 문서를 architecture/spec semantic source로 사용하지 않는다.
- 실제 개선 상태는 `docs/process/doc_improvement_tasks.md`에서 추적한다.
- 현재 유효한 process control-plane은 아래 문서를 우선한다.
  - `docs/process/spec_driven_development_workflow.md`
  - `docs/process/project_roadmap.md`
  - `docs/process/contribution_and_review_guide.md`
  - `docs/process/naming_convention.md`

## Maintainer Guidance

새로운 리뷰를 추가할 때는 이 문서에 장문 분석을 누적하지 않는다.  
대신 필요한 경우:

1. 새 review note를 archive에 저장하고
2. actionable item은 `doc_improvement_tasks.md`로 옮기며
3. 실제 정책 변화는 workflow/roadmap/governance 문서에 반영한다.
